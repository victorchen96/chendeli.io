📚 模组简介 Overview

竞技平衡 - 为多人对战而生

BBG (Better Balanced Game) 是文明6最流行的竞技平衡 Mod，由国际多人对战社区维护。Mod 分为两个部分：

  • Classic BBG: 核心平衡性调整，重做几乎所有原版文明的能力数值，消除过强/过弱文明
  • BBG Expanded: 可选扩展包，新增4个全新文明和6个新领袖，为竞技对战增加多样性

BBG 的设计哲学是让每个文明都有独特的竞争优势，同时避免任何文明过于碾压或无法使用。

💻 安装指南 Installation

Steam Workshop 安装

  1. 搜索 "Better Balanced Game" 或 "BBG"
  2. 订阅 Classic BBG（必须）和 BBG Expanded（可选）
  3. 在游戏大厅中启用 BBG 模组
  4. 多人对战时所有玩家需安装相同版本

注意事项

  • BBG 与大多数 UI Mod 兼容
  • 不建议与其他平衡 Mod（如 NQ Mod）同时使用
  • BBG Expanded 需要 Classic BBG 作为前置
